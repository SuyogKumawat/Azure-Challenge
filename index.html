GIT and GITHUB Basics

GIT

  • Git is distributed VCS we can edit the files without network connection and later sync with server when connection becomes available
  • Working Tree refer as set of directories and files.
  • branch is named as series of commits
  • blob object contains an ordinary file.

GIT COMMANDS

  • git push
  • git --version
  • git config --global user.name "USER_NAME"
  • git config --global user.email "USER_EMAIL"
  • git config --list
  • git init
  • git status
  • git commit index.html -m "Create an empty index.html file"
  • git pull
  • git reset --hard

GITHUB

  • GitHub is a cloud platform that uses Git as its core technology

Features of GITHUB

  • Issues
  • Discussions
  • Pull requests
  • Notifications
  • Labels
  • Actions
  • Forks
  • Projects
